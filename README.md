# Tic Tac Toe Lab

This project is a basic Tic Tac Toe game made with HTML, CSS, and JavaScript.

## Features

- Player X always starts.
- Two players can take turns.
- The game checks for wins after every move.
- The game checks for a tie when all 9 squares are filled.
- The score is saved for Player X and Player O.
- The New Game button clears only the board.
- The Reset button clears both the board and the scores.
- The game has a computer mode that uses the Minimax algorithm to choose the best move.
